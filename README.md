# Student Performance Analysis

Academic performance analysis using Python and Power BI dashboards.

## Project Overview
This project analyzes student data to understand the impact of attendance, gender, and study patterns on academic results. Dashboards and charts are used to visualize trends and insights.

## Tools Used
- Python
- Power BI
- Excel

## Key Features
- Attendance Impact on Marks
- Gender vs Average Scores
- Grade Distribution Charts
- Subject Score Trends
